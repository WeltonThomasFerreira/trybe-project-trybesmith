## Rodando o projeto
Esse projeto utiliza variáveis de ambiente e você pode utilizar um arquivo `.env` para isso.

### Via docker + node:
```
npm run compose:up
```

### Via docker:
```
sudo sh ./scripts/compose-up.sh
```
